## Recipe Sharing Plateform

## About
A responsive recipe sharing application built with React, Tailwind CSS, Context API, and React Router for smooth client-side navigation. Dynamic recipe pages are implemented using `useParams`.

## Features
- Responsive user interface
- Browse multiple recipes
- Dynamic recipe detail pages
- Submit Recipe page
- Working Search Bar
- Category Filter
- State management with Context API
- Navigation with React Router
- Styled with Tailwind CSS

## Tech Stack
- React.js
- React Router
- Tailwind CSS
- Context API
- JavaScript (ES6)

## Setup & Installation
Clone the repository:

git clone [<repo-url>](https://github.com/RukhsarYaqoob/Recipe-Sharing-Website.git)

Install dependencies:

npm install

Run the development server:

npm run dev
